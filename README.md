The folder [#172](https://github.com/JAgbanwa/Erdos-Problems/tree/main/%23172) gives exposition for the question

 Is it true that in any finite colouring of $\mathbb{N}$ there exist arbitrarily large finite $A$ such that all sums and products of distinct elements in $A$ are the same colour?

The checked Lean project lives in `ErdosProblems/` and builds from the repository root with `lake build`.
