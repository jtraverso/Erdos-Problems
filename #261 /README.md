# Problem 261

See `solution.tex` for the write-up.

Main unconditional conclusions:

- Infinitely many positive integers \(n\) have a finite distinct representation.
- The explicit family is \(n=2^{r+1}-r-2\), \(r\geq 2\).
- The question of whether every \(n\) works is reduced to a concrete residual recurrence; no all-\(n\) claim is made.
- The rational number \(x=1\) has at least \(2^{\aleph_0}\) distinct infinite representations.
