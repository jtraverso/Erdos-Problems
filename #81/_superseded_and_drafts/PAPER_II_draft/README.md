# Paper IV — Official Author Preprint v1.0

**Title:** *Erdős #81 for Chordal Graphs (r=3): Cover-First Proof*

Paper IV is the installment that announces the proof of the asymptotic chordal case of Erdős Problem #81:

```text
max{cp(G): G chordal, |V(G)| = n} = (1/6 + o(1)) n^2.
```
