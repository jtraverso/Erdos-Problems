# Paper II v0.8 — Submission Claims Matrix

| Topic | Safe wording |
|---|---|
| Symmetrization | “We apply the fractional clone-symmetrization lemma of Balogh et al.” |
| Structural contribution | “We prove that the constrained chordal process terminates in complete split graphs.” |
| Threshold consequence | “The same extremal value is attained within threshold graphs.” |
| Orbit program | “The full packing LP on a complete split graph is exactly equivalent to a two-variable orbit LP.” |
| Finite \(K_r\) result | “We determine the exact finite maximum of \(\Phi_r^*\) over chordal graphs.” |
| Integral theorem | “The integral first-order result uses standard fractional-to-integral packing.” |
| Algorithm | “For every fixed \(r\), a complete-split fractional extremizer is constructible in polynomial time.” |
| Computation | “Scripts are supplementary regression tests and not proof premises.” |
| Novelty | “No prior chordal complete-split reduction was found in a targeted search; definitive review remains pending.” |

## Avoid

- claiming the Balogh symmetrization as new;
- claiming polynomial time jointly in \((n,r)\);
- claiming the algorithm constructs the integral optimum;
- describing numerical verification as proof;
- claiming stability from the exact extremizer theorem.
