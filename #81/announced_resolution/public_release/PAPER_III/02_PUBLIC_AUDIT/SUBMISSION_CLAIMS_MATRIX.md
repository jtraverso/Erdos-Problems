# Paper III v1.0.4 — Submission Claims Matrix

| Topic | Safe wording |
|---|---|
| Main theorem | “There exists an absolute constant \(C\) such that every split graph satisfies \(|E|-2\nu_3\le n^2/6+Cn\).” |
| Clique partitions | “Consequently, \(\operatorname{cp}(G)\le n^2/6+Cn\) for split graphs.” |
| Sharpness | “The Erdős--Ordman--Zalcstein complete-split, indeed threshold, construction shows that the leading constant \(1/6\) is sharp.” |
| Integrality gap | “The proof does not establish \(\nu_3^*-\nu_3=O(n)\) uniformly.” |
| Effectivity | “The near-extremal corridor is effective; the global constant remains non-effective because of two external asymptotic thresholds.” |
| Computation | “Scripts are supplementary regression tests and are not logical premises.” |
| Review status | “The manuscript has undergone extensive AI-assisted adversarial checking but has not yet been independently peer-reviewed.” |

## Avoid

- claiming an explicit global value of \(C\);
- claiming a universal linear fractional-to-integral gap;
- describing computational enumeration as proof;
- claiming independent external or journal review;
- attributing the same lower-bound construction inconsistently across the paper series.
